# Custom-Medical-Drone
<img width="659" height="447" alt="image" src="https://github.com/user-attachments/assets/4105fb12-7df0-4291-abd8-e9191c15a4aa" />
This is my hack club project to build a medical drone from scratch with a custom IMU with Arduino &amp; MPU6050. The drone will also have a Raspberry PI with a camera in order to detect humans in the view to notify the user when to drop the medication. This is a new approach as the quick travel time of the drone can result in very fast delivery.
