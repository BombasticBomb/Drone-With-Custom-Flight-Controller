# Custom-Medical-Drone
<img width="1089" height="632" alt="model" src="https://github.com/user-attachments/assets/bb9c271a-90ce-41d8-b988-920256057338" />

This is my hack club project to build a medical drone from scratch with a custom IMU with Arduino &amp; MPU6050. The drone will also have a Raspberry PI with a camera for basic machine learning in order to detect humans in the view to notify the user when to drop the medication. This is a new approach, as the quick travel time of the drone can result in very fast delivery.
